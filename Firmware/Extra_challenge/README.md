# Extra challenges
In case the other challenges are finished, you can try the following additional challenges


## Challenge 1
This challenge will be to add the functionality of acquiring data from the SHT31 sensor periodically instead of doing it in single mode. More precisely, you will need to check the datasheet of the STH31 sensor and implement new I2C commands.

## Challenge 2
The PCB has a NTC resistor. Configure the ADC and make the needed calculations to measure the temperature using that sensor.

## Challenge 3
The PCB also has an RGB LED (WS2812C). Configure the SPI so that the color of that LED can be controlled.
